# BUSLA — Smart School Transportation

Interactive demo of an AI-powered school transportation management system.

## Demo Features

- **Live bus tracking** — Animated buses moving along routes on an interactive map
- **Student stops** — Each stop shows student count and pickup status
- **Real-time ETA** — Travel times update as buses move
- **Onboard students** — See which students are on each bus
- **Trip management** — Sidebar shows all active trips with progress
- **Route visualization** — Multiple bus routes with color-coded paths

## How to Use

```bash
# Open directly in browser
open index.html

# Or serve locally
npx serve .
python -m http.server 8000
```
